HTML Elements

HTML element is defined by a starting tag with some content and an ending tag.

Concept of HTML Elements

All the content of the containt posses between the stating and ending tag.

< tagname > Content Here < /tagname >

Examples of some HTML elements:

< h1 > This is a heading. < /h1 >


< p > This is a paragraph. < /p >

Starting Tag Element Content Ending Tag
< h1 > This is a heading. < /h1 >
< p > This is a paragraph. < /p >
< br > N/A N/A

#Note: Some HTML elements have no content (like the < hr > or < br > element). These elements are called empty elements. Empty elements do not have an end tag!

Nested HTML Elements

HTML Element can contain other Elements.

All HTML documents consist of nested HTML elements.

The following example contains four HTML elements (<html>, <body>, <h1> and <p>):

Example

< !DOCTYPE html >

< html >
< Head >
< title > HTML Tutorial < /title >
< head >
< body >

< h1 > This is a heading < /h1 >
< p > This is a paragraph. < /p >

< /body >
< /html >

Example Explanation

  • The <html> element is the root element and it defines the whole HTML document.
  • It has a start tag <html> and an end tag </html>.
  • Then, inside the <html> element there is a <body element.
  • The <body> element defines the document's body.
  • It has a start tag <body> and an end tag </body>.
  • Then, inside the <body> element there are two other elements: <h1> and <p>.
  • The <h1> element defines a heading.
  • It has a start tag <h1> and an end tag </h1>.
  • The <p> element defines a paragraph.
  • It has a start tag <p> and an end tag </p>.

Never Skip the End Tag

Some HTML elements will display correctly, but if you forget the end tag other HTML element will not work properly.

< html >
< body >

< h1 > This is a heading
< p > This is a paragraph.

< /body >
< /html >

#Note: In output above code will show error because tags are not ended properly. Unexpected results and errors may occur if you forget the end tag! So it is must to end the tags properly

Empty HTML Elements

HTML elements with no content are called empty elements.

The <br< tag defines a line break, and is an empty element without a closing tag

Example

<h1> This is a heading <br> This is a paragraph.</h1>
